# Pilot 0.0.2 Plan

## Goal

Keep Pilot near 100M parameters, but make the training recipe less naive than
0.0.1.

## Model

- Llama-compatible causal LM
- About 103M parameters
- 18 layers
- Hidden size 640
- 10 attention heads
- 5 KV heads
- Intermediate size 2048
- Context length 2048 for training, 4096 max positions
- Custom byte-level BPE tokenizer with 16k vocabulary
- Tied input/output embeddings

## Training

Stage 1: pretrain on English/Spanish text plus a small amount of formatted chat.

Target: about 500M training tokens.

Stage 2: supervised chat tuning with labels masked so loss applies only to the
assistant answer, not the system/user prompt.

## Quality Gate

Before exporting to GGUF or importing into LM Studio, run:

```powershell
python scripts\eval_pilot_0_0_2.py --hf-dir runs\pilot_0_0_2_sft\hf_last
```

The first useful target is not intelligence. The first target is stable behavior
on basic prompts:

- `hola`
- `como estas?`
- `quien eres?`
- simple translation
- simple summary
- simple Python code
